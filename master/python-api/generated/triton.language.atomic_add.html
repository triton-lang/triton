triton.language.atomic_add¶

triton.language.atomic_add(pointer, val, mask=None)¶

Performs an atomic add at the memory location specified by pointer.

Return the data stored at pointer before the atomic operation.

Parameters

  • pointer (Block of dtype=triton.PointerDType) – The memory locations to apply add.

  • val (Block of dtype=`pointer.dtype.element_ty`) – The values to add in the atomic object.

  • mask (Block of triton.int1, optional) – If mask[idx] is false, do not apply add.
